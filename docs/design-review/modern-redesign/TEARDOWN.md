# Design Teardown — لماذا كان التصميم السابق يحتاج إعادة تصميم

التصميم السابق (الكلاسيكي/الزخرفي) كان مرتّباً ووظيفياً، لكنه لم يصل إلى
«modern luxury». الأسباب:

1. **زخرفة تقليدية زائدة:** فواصل ذهبية بقلوب، زخارف أركان على البطاقة،
   إطار مزدوج، نجوم/أزهار → يعطي إحساس «بطاقة زفاف قديمة» أو قالب جاهز.
2. **الذهب كلون غالب:** استُخدم الذهب كعنصر رئيسي (إطارات، عناوين، حدود) بدل
   أن يكون لمسة خفيفة → ثقيل وكلاسيكي.
3. **تايبوغرافي كاليغرافي تقليدي:** خط Aref Ruqaa جميل لكنه أكبر إشارة
   «كلاسيكية»؛ لا يناسب الاتجاه العصري.
4. **تماثل وزحام:** كل قسم بعنوان محاط بحلية مزدوجة، كل شيء متمركز ومزخرف →
   يفتقر للهدوء والمساحة البيضاء المحسوبة.
5. **تدرّجات وتوهّجات كثيرة:** خلفيات بعدة radial gradients → بعيدة عن نظافة
   الـ modern luxury.
6. **البطاقة:** إطار ذهبي مزدوج + زخارف → ليست «Instagram story» حديثة قابلة
   للنشر كتصميم احترافي.
7. **الحركة:** lift/sheen على الأزرار، حركات كثيرة → أقرب للـ «template» منها
   للـ editorial الهادئ.

## المبادئ التي قاد عليها إعادة التصميم
- **تباين سينمائي:** افتتاح/هيرو/فوتر داكن دافئ، جسم فاتح editorial.
- **تايبوغرافي قوي:** Cairo (عربي، أوزان خفيفة بمقاييس كبيرة) + Playfair Display
  للأرقام/اللاتيني → إحساس مجلة فاخرة.
- **ألوان مكتومة:** ivory / charcoal / burgundy عميق / muted rose، والذهب
  **خيوط رفيعة ولمسات فقط**.
- **مساحة بيضاء سخية**، خطوط فاصلة رفيعة بدل الزخارف.
- **حركة هادئة:** fade + slide خفيف فقط.
- **بطاقة minimal:** keyline رفيع، الرسالة هي البطل، بلا إطار ثقيل.
