# Modern Luxury Redesign — elzant.com

تحويل الواجهة من كلاسيكي/زخرفي إلى **modern luxury · editorial / cinematic**.

## نظام التصميم
- **الخطوط (self-hosted):** Cairo للعربية وواجهة الاستخدام (أوزان 300–700)،
  Playfair Display للأرقام/اللاتيني (التواريخ، elzant.com، الرموز). أُسقط Aref Ruqaa.
- **الألوان:** ivory فاتح دافئ (`#f5efe6`) للجسم، ink داكن دافئ (`#1b1412`) للافتتاح/الهيرو/الفوتر،
  burgundy عميق (`#581a29`) أساسي، muted rose (`#c68f95`) لمسة، soft gold (`#ab8d56`) خيوط رفيعة فقط،
  charcoal (`#271f1c`) للنص.
- **الحركة:** `up` (دخول هادئ للهيرو) + `reveal` (ظهور عند التمرير). لا confetti/glitter.

## ما تغيّر بصرياً (قسماً بقسم)
| القسم | قبل | بعد |
|---|---|---|
| الافتتاح | ظرف كرتوني + ختم شمع | شاشة افتتاح داكنة minimal: أسماء مكدّسة، خط ذهبي رفيع، تاريخ Playfair، زر «الدخول» |
| Hero | كريمي مزخرف + شعار خواتم | داكن سينمائي: تايبوغرافي كبير، عدّاد **مدمج** نظيف، portrait editorial، CTA واضح |
| العدّاد | بطاقات داكنة بحدود ذهبية | وحدات minimal بأرقام كبيرة وفواصل رفيعة (مدمج في الهيرو) |
| التفاصيل | لوحة بزخارف | نص editorial، تاريخ Playfair، خط فاصل رفيع |
| النموذج | لوحة كريمية | تجربة كتابة رسالة: حقول كبيرة نظيفة، microcopy دافئ |
| الجدار | بطاقات بعلامات اقتباس ذهبية | بطاقات surface نظيفة، مسافات سخية، تصلح للطويل والقصير |
| المشاركة | QR بإطار ذهبي | QR داخل card نظيفة + زر واتساب مسطّح |
| البطاقة | إطار ذهبي مزدوج + زخارف | minimal: keyline رفيع، الرسالة هي البطل، أسماء Cairo + Playfair |
| OG | ذهبية مزخرفة | داكنة سينمائية: «WEDDING INVITATION» + الأسماء + التاريخ فقط |

## لماذا الجديد أفضل
- يبدو **expensive / editorial** لا «template/AI generic».
- تايبوغرافي ومساحة بيضاء يقودان الإحساس، لا الزخرفة.
- التباين الداكن/الفاتح يعطي إحساساً سينمائياً وثقة.
- البطاقة الجديدة قابلة فعلاً للنشر على Instagram Story / واتساب كتصميم محترف.

## الأصول
- `og-image-preview.png` — صورة Open Graph النهائية (1200×630) كما تظهر عند مشاركة الرابط.

> **ملاحظة عن لقطات الواجهة:** أدوات المعاينة في هذه البيئة تُرجِع اللقطات إلى
> المساعد (وقد رُوجعت بصرياً على mobile 375 و desktop 1280)، لكنها لا تكتب صور
> المتصفّح إلى القرص آلياً. لإعادة توليد اللقطات: شغّل `python manage.py runserver`
> وافتح الموقع. تصدير البطاقة PNG اختُبر فعلياً ونتيجته 2160×2700 بعربية سليمة.
