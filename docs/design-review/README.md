# مراجعة التصميم البصري — elzant.com

مراجعة وتحسين بصري للواجهة باستخدام إضافة `frontend-design`، بهدف إزالة الطابع
«العام المولّد آلياً» ورفع الواجهة إلى مستوى دعوة زفاف فاخرة — **دون أي تغيير في
المنطق الخلفي (Django forms / moderation / card export)**.

الصور: `before/` (قبل) و `after/` (بعد) — لكل من الجوال (390px) والحاسوب (1440px)،
إضافة إلى لقطات مقرّبة للهيرو والعدّاد والبطاقة المُصدّرة فعلياً.

## التشخيص (قبل)

| العلامة العامة | التشخيص |
|---|---|
| العدّاد | أربعة صناديق متطابقة «رقم كبير + تسمية» — النمط الافتراضي الأكثر شيوعاً |
| القلوب | قلب متكرّر في الهيرو/الفوتر/الجدار/المظروف + إيموجي 🤍 — زخرفة غربية عامة |
| الإيقاع | كل قسم متمركز بنفس عنوان الخطّين الذهبيين — لا تنوّع بنيوي |
| الذهب | لون واحد مسطّح بلا لمعان معدني |
| البورتريه | دائرة نائبة عامة بحلقتين |

## التحسينات المطبّقة

- **التوقيع البصري:** نجمة ثمانية ذهبية (نجمة/najmah) — زخرفة هندسية عربية أصيلة
  حلّت محل كل القلوب في الموقع والبطاقة والمظروف وصورة OG.
- **ذهب معدني (`.text-gilt`):** تدرّج شامبانيا→ذهبي→برونزي على اللمسات (الـ ampersand،
  العلامة النصية، علامة الاقتباس).
- **إطار الدعوة (`.page-frame`):** خيط ذهبي رفيع يحيط بإطار العرض — يربط الموقع
  بالبطاقة وصورة OG بصرياً.
- **نسيج ورق الدعوة:** شبكة معيّنات ذهبية باهتة جداً خلف الصفحة.
- **العدّاد:** أُعيد تصميمه كـ«ساعة واحدة» موصولة بفواصل ذهبية بدل الصناديق المنفصلة.
- **إيقاع طباعي:** سطر تمهيدي (eyebrow) مفصول فوق كل عنوان، إبراز خط Aref Ruqaa الخطّي.
- **عمق اللوحات:** إضاءة علوية خفيفة + حافة ذهبية معدنية.
- **البورتريه النائب:** مونوغرام أنيق (نجمة + حلقتان متشابكتان) بدل الحلقتين العامتين.

## الاختبارات بعد التعديل (كلها ناجحة)

- `npm run css:build` ✓
- `npm run assets:build` ✓ (إعادة توليد og-image.png و qr.svg)
- `python manage.py check` ✓ — لا مشاكل
- `python manage.py collectstatic --noinput` ✓
- **إرسال تهنئة:** POST صحيح → تحويل لصفحة الشكر، تُحفظ كـ `pending` (تخضع للمراجعة) ✓
- **المراجعة:** الجدار يعرض المعتمدة فقط، والمعلّقة مخفيّة ✓
- **honeypot / التحقق:** الرسائل المزيّفة والفارغة مرفوضة دون حفظ ✓
- **تصدير البطاقة PNG:** يعمل، 2160×2700، الخطوط العربية مضمّنة بشكل سليم ✓

> لم تُمسّ أي ملفات منطق خلفي (views/models/urls). التعديلات حصراً على القوالب،
> ونظام تصميم Tailwind (`tailwind/input.css`)، وأصول الصور.
